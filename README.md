# Woo Live Auctions by CyberCraft

## Brief
Woo Live Auctions supercharges WooCommerce with high-energy bidding that keeps shoppers glued to the page. Deliver real-time excitement, effortless auction management, and a premium experience out of the box.

## Description
Woo Live Auctions transforms any WooCommerce store into a broadcast-worthy auction floor. From lightning-fast AJAX updates to smart proxy bidding and automated lifecycle management, every detail is engineered to convert browsers into bidders. Built by the product team at CyberCraft, the plugin blends modern UX, transparent freemium limits, and actionable admin tooling so you can launch auctions in minutes.

## Features
* **Realtime bidding theater** – AJAX-powered increments, countdown refreshes, and no page reloads.
* **Smart proxy bidding** – Free tier handles light proxy competition; Pro unlocks unlimited auto-duels.
* **Lifecycle automation** – Cron-driven start, pause, and finish logic keeps auctions on schedule.
* **Bidder communication** – Targeted emails for outbid, ending soon, winners, proxy wars, and more.
* **Premium UI everywhere** – Responsive layouts, bid history scroller, watchlists, and front-end buy now.
* **Admin clarity** – Dashboard notices, analytics hooks, and manual override controls give you command.

## Why This Is More Than You Need
Most auction plugins stop at listing items. Woo Live Auctions choreographs the entire show: automated scheduling, intelligent notifications, unbiased freemium messaging, and monetization prompts for shop owners. You get 5-star velocity without a custom dev sprint.

## Activation
1. Upload or install from the WordPress plugins screen.
2. Activate the plugin.
3. Visit **WooCommerce → Settings → Auctions** to review defaults.
4. Create a product and switch the product type to **Auction**.
5. Set start/end times, increments, reserve, and optional buy-now pricing.
6. Publish, promote, and watch the bids climb.
